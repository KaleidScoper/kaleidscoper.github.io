# 贺新郎·一个人的西湖之旅
# 沈梦核〔现代〕
# 五月杭州旅。客西湖、苏堤内外，太多情侣。深握人儿温柔手，各有千般思绪。鱼水共、桃花向暮。自忖平生恩和爱，误了谁、又被谁耽误。风接送，伤心渡。
# 怦然昨日无穷句。怎如今、从容一改，满怀犹豫。慷慨歌声青春气，尽让光阴窃取。问知己、何时天与。但恨红尘游戏短，结局为、同作鸳鸯古。出字幕，放金缕。

# 满江红·燕子楼中

# 燕子楼中，又捱过、几番秋色。
# 相思处、青年如梦，乘鸾仙阙。
# 肌玉暗消衣带缓，泪珠斜透花钿侧。
# 最无端、蕉影上窗纱，青灯歇。
曲池合 高台灭 人间事 何堪说

世态便如翻覆雨 妾身元是分明月
# 笑乐昌、一段好风流，菱花缺。
