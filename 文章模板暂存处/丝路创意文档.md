# 游戏创意文档：丝绸之路 · 双主角RPG

## 一、核心概念
- **类型**：线性流程 + 轻度RPG（类似《刺客信条：起源》的主线感受，但更简洁）  
- **主题**：通过丝绸之路，展现东西方文明在动荡年代中的交流与冲突。  
- **玩法基调**：探索 + 见证 + 轻度战斗 + 历史体验  

## 二、双主角设定
- **主角一（东方）**  
  - 出发地：扬州（唐帝国东南重镇、繁华商贸港口）  
  - 目标：前往长安，探寻家族秘密/真相；在战乱爆发后被卷入动荡。  
  - 路线：扬州 → 洛阳/长安 → 西域 → 草原/中亚  
  - 核心体验：见证唐帝国的繁荣与衰败。

- **主角二（西方）**  
  - 出发地：大马士革（阿拔斯王朝初期首都巴格达的前身重镇）  
  - 目标：前往巴格达，再踏上丝绸之路，追寻遗产或某种秘密。  
  - 路线：大马士革 → 巴格达 → 撒马尔罕/碎叶 → 西域边境  
  - 核心体验：在新兴的阿拔斯王朝中见证政治斗争与宗教纷争。

- **双主角关系**  
  - 两人互不相识，但经历相似，剧情采用插叙切换。  
  - 在边境交汇处（例如碎叶城、撒马尔罕或敦煌）相遇。  
  - 相遇时因误会或冲突展开对峙，后逐渐理解彼此。

## 三、历史背景
- **时间选择**：8世纪中期（唐代安史之乱前后，阿拔斯王朝初立）  
- **东亚**：  
  - 唐朝盛极而衰，长安仍为繁华之都，但即将爆发安史之乱。  
  - 长安、洛阳等地成为动乱核心，丝绸之路逐渐受阻。  
- **西亚**：  
  - 阿拔斯推翻倭马亚，迁都巴格达，正值政权更迭的关键时期。  
  - 大马士革失势，但仍是文化和宗教的重要中心。  

- **战乱点**：  
  - 唐：安史之乱（755年起）  
  - 西亚：阿拔斯革命（750年前后），权力斗争余波未平  
- **相对稳定区**：  
  - 中亚粟特商人城邦（撒马尔罕、碎叶等）  
  - 吐蕃、西域部分绿洲  

## 四、路线与景观
- **东方主角**：扬州水乡风光 → 长安宫阙 → 河西走廊 → 敦煌壁画 → 西域戈壁 → 昆仑山北麓自然奇观  
- **西方主角**：大马士革清真寺与市集 → 巴格达智慧宫（设想） → 波斯平原 → 撒马尔罕繁荣的商贸 → 天山 → 荒漠绿洲  

## 五、关键剧情节点
1. **序章**：两地同时开篇，奠定主角动机（家族秘密/遗产）。  
2. **都城高潮**：主角一抵达长安，主角二抵达巴格达，感受帝国繁荣。  
3. **动荡爆发**：安史之乱 + 阿拔斯政权整合，推动剧情转向。  
4. **边境相遇**：在碎叶/撒马尔罕，双方因误解冲突，继而被更大危机迫使合作。  
5. **终章**：故事未必有“完美结局”，更强调命运洪流下的个人抉择。  

## 六、BOSS与冲突设定
- **历史人物**（非全为敌人，但可设计为“冲突事件”）：  
  - 东方：安禄山、郭子仪、唐玄宗、杨国忠  
  - 西方：阿布·阿拔斯、曼苏尔（阿拔斯王朝早期哈里发）  
- **虚构敌人**：商旅强盗、雇佣兵团、地方军阀，突出荒漠与边境的危险感。  

## 七、艺术与美术风格
- **核心理念**：强调“文明对比”与“自然奇观”  
  - 东方：水乡、宫廷、壁画、佛窟  
  - 西方：清真寺、波斯园林、学术殿堂、集市  
  - 边境：荒漠戈壁、雪山天山、绿洲城邦  

## 八、差异化卖点
- 双主角插叙式剧情，玩家体验东西方两种文化。  
- 丝绸之路大地图串联，从繁华都城到荒凉边境，形成叙事对比。  
- 强调“见证历史”的沉浸感，而不是传统的“改变历史”。  

## 九、需要重点考据的时代范围
- **时间轴核心范围**：公元740年 — 770年  
  - **前段**：唐玄宗天宝年间（繁盛极致 → 安史之乱爆发 755年）  
  - **中段**：安史之乱持续（755—763），长安、洛阳失陷  
  - **后段**：肃宗、代宗收复中原，唐朝衰退迹象明显  
  - **西方**：倭马亚王朝末期 → 阿拔斯革命（750年） → 曼苏尔建都巴格达（762年）

## 十、需要重点研究的国家与地区
- **东方（中国与周边）**  
  - 唐帝国（首都长安，东南重镇扬州）  
  - 吐蕃王朝（对唐西部与西域形成压力）  
  - 契丹、回纥（北方势力，安史之乱中回纥成为盟友）  
  - 西域绿洲：龟兹、于阗、碎叶、敦煌  

- **西方（伊斯兰世界与周边）**  
  - 倭马亚王朝（大马士革为都，750年被推翻）  
  - 阿拔斯王朝（巴格达建都后成为新中心）  
  - 撒马尔罕、布哈拉（粟特商人主导的丝路重镇）  
  - 波斯（萨珊余韵尚存，阿拉伯化、伊斯兰化进程加快）

## 十一、需要重点考据的城市
- **唐帝国**：长安、洛阳、扬州、敦煌、碎叶  
- **西亚**：大马士革、巴格达、撒马尔罕、布哈拉  
- **自然关口与奇观**：  
  - 河西走廊（祁连山、敦煌莫高窟）  
  - 昆仑山北麓  
  - 天山南北路（丝路中亚段核心）  

## 十二、相关重要人物（可作为参考/出场/虚化角色）
- **东方**  
  - 唐玄宗（盛世与衰落的象征）  
  - 杨贵妃、杨国忠（权臣，安史之乱前的导火索）  
  - 安禄山（叛乱首领）  
  - 郭子仪（收复中原的名将）  
  - 吐蕃赞普赤德祖赞（与唐对峙的强敌）

- **西方**  
  - 阿布·阿拔斯（阿拔斯王朝开国）  
  - 曼苏尔（第二任哈里发，762年建巴格达）  
  - 倭马亚残余势力（逃往西班牙，形成安达卢斯）  
  - 波斯学者、粟特商人群体（文化与贸易核心）

## 十三、重大历史事件
- **东方**  
  - 安史之乱（755—763）  
  - 回纥出兵助唐  
  - 长安、洛阳陷落，唐帝国元气大伤  
- **西方**  
  - 阿拔斯革命（750年）推翻倭马亚  
  - 巴格达建都（762年），成为伊斯兰世界学术与商业中心  
  - 撒马尔罕等粟特城邦在东西方交流中的崛起  

## 十四、推荐阅读著作
### 中国史相关
- 《旧唐书》《新唐书》（二十四史原典，安史之乱与对外关系详载）  
- 《资治通鉴》卷 217—222（详细记述安史之乱）  
- 《大唐西域记》（玄奘著，虽早于本时代，但对丝路地理文化描写极详）  
- 黄永年：《唐代政治史稿》  
- 陈寅恪：《唐代政治史述论稿》  

### 丝绸之路与中亚
- 《丝绸之路：一部全新的世界史》（彼得·弗兰科潘，现代学术写作，宏观叙事）  
- 《丝绸之路通史》（荣新江主编，中国学界代表性成果）  
- 《粟特人在中国》（荣新江，研究粟特商人活动与中西交流）  

### 伊斯兰与中东史
- 《剑桥伊斯兰史·第一卷》（涵盖倭马亚到阿拔斯早期历史）  
- Hugh Kennedy: *The Early Abbasid Caliphate*  
- Patricia Crone: *God’s Caliph*（阿拔斯革命背景）  
- 马坚译《古兰经》（了解伊斯兰思想氛围）

### 辅助阅读（氛围营造）
- 《敦煌遗书选编》  
- 《天方夜谭》（虽为后世整理，但可提供中东文化氛围灵感）  
- 《唐诗三百首》：李白、杜甫等诗人作品对社会氛围描绘极佳  

## 十五、研究路径建议
1. **宏观历史**：先通读《资治通鉴》相关章节与《剑桥伊斯兰史》，掌握东西方大势。  
2. **区域重点**：分别研究唐代中原+西域，以及阿拔斯初期+粟特城邦。  
3. **微观文化**：参考敦煌文书、唐诗、中东市集描写，增强生活氛围细节。  
4. **人物与事件串联**：在真实历史事件中为双主角安排“见证”或“卷入”的情境。  
5. **核心团队（20–40 人）**：
  - 制作人/执行制作人 ×1–2  
  - 主策划 + 关卡策划 ×2–3  
  - 编剧/叙事设计 ×2  
  - 美术团队（概念、建模、贴图、动画）×8–15  
  - 程序团队（引擎、AI、战斗、UI、网络）×6–10  
  - 音乐与音效 ×2–3  
  - 历史顾问团队（兼职或外包）  
  - QA 测试团队 ×4–6
