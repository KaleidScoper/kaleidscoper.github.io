# 零

暂时存放我的虚构写作点子。

# 一 如花绽放 如水沸腾
一种新出现在市场的农具长的像四棱细石柱，只需短暂插入土中，便可把一大片地翻好，好像滚水上的浪花。不过，最近不知怎的买不到了。数月之后的夜色中，一根刻满符文的方尖碑被运上了一艘浮空战舰。

# 二 
武试上有少年能以低温为武器战斗，反制起来很简单，尝试不接触他就好了。想必他很难被选进军队。六月份，有少年将军带领着一支身着棉甲的军队现身了。人们抬起头，看见浅灰色的云。

# 三 蓑衣花
如果有足够的史料和物品的残骸，这个少年能将任意一物在历史上的任一时期带到现实，代价是一点自己的时间。没人见过他做到这一点，或许是他编造的。
他很喜欢记日记。他的兴趣爱好广泛得异常。不过，他似乎不在意忘掉旧的知识去学习新的。
他的国家遭到入侵。交通要道上，有一座颓坯的关隘，国家尚且分裂时的工事。不过敌军到达时，迎接他们的是一座雄关和千军万马。每一位士兵长的都很像，老幼不一。然后矢发如雨，间以电火冰石。城破。关后小山上有藏经阁，或曰弃置多年。阁中有一白发老者，纸页、白发铺满地板，一时间不能分辨。面目似先前兵卒。立斩之。

# 四
这个世界的每一种魔法都可以靠环形引导，从戒指到手镯，不一而足，每一枚环可以引导一种特定的魔法。就在刚刚，主角团新结识了一位右臂戴满镯子的女孩。
这个世界的人们都很好奇，地平线上的大环形山会引导出什么样的魔法。

# 五 
倾倒的书架下，有少女倒在血泊中，身上布满了残忍的刀伤。相较于她的死，正在被焚毁的图书馆是一笔较小的损失。
