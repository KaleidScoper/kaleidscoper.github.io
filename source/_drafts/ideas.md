# 零

暂时存放我的虚构写作点子。

# 一 如花绽放 如水沸腾
一种新出现在市场的农具长的像四棱细石柱，只需短暂插入土中，便可把一大片地翻好，好像滚水上的浪花。不过，最近不知怎的买不到了。数月之后的夜色中，一根刻满符文的方尖碑被运上了一艘浮空战舰。

# 二 
武试上有少年能以低温为武器战斗，反制起来很简单，尝试不接触他就好了。想必他很难被选进军队。六月份，有少年将军带领着一支身着棉甲的军队现身了。人们抬起头，看见浅灰色的云。

# 三 蓑衣花
时间并不对每一个人都公平，而是格外偏爱他。如果有那样一个天平，他的一秒会比旁人的千百年都要沉重。
因此，如果有足够的史料和对应物品的残骸，这个孩子能将那东西在历史上的“任一瞬间”带回今天，代价仅仅是一秒自己的时间。
像所有小孩子一样，他很喜欢记日记。他的兴趣爱好也广泛得异常，而且喜新厌旧。
有一座颓坯的关隘设在国家腹地的险要之处。敌军到达时，迎接他们的是一座雄关和千军万马，每一位士兵长的都很像，老幼不一。
然后，矢发如雨，间以电火冰石。
城破。有藏经阁，阁中有一白发老者，纸页、白发铺满地板，仿佛雪后的荒原。

> “蓑衣花刀”是中式厨房的一种技法，利用特殊的切法使被切的食材分为大量——可能是数百片——极薄的薄片，但同时还能保持连接不会散开。

# 四
这个世界的每一种魔法都靠环形引导，从戒指到手镯，不一而足，每一枚环都可以引导一种特定的魔法。就在刚刚，我们新结识了一位右臂戴满镯子的女孩。
这个世界的人们都很好奇，地平线上的大环形山会引导出什么样的魔法。

# 五 
倾倒的书架下，有少女倒在血泊中，身上布满了残忍的刀伤。相较于她的死，正在被焚毁的图书馆是一笔较小的损失。

# 六
百战不殆的明证：在晚年授予离退休大将军的勋章，证明此人未尝败绩。据说其中封存有颁布此章的皇帝的一滴血，用特殊的魔法将其与皇室持有的血样对比，能验明真伪，故而无人敢伪造。不过，眼下它正佩戴在一位少年身上。

# 七
法师是一种非常珍贵的资源——不论死活。如果你能恰当地保存一个法师的尸体（至少包含一个头、完整脊柱和与之连接的一个脏器/肢体，取决于具体需求），你仍有办法利用它来激发此人生前掌握的某些或者全部魔法。最近有人看到帝都城墙上的火药大炮被拆除了。

# 八
今夜很晴朗，天上没有一丝云。借着满月的光，你能看清这座小岛的全貌。上有一台，台上有亭，亭下有茶盏冒着热气。然后你看到了如棉絮的海面，无边无际。
