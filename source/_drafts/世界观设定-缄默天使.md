# 世界观设定：缄默天使

> 本文件为游戏世界观的框架性文档，包含核心背景、科技设定、飞船分类与状态谱系。文档内容为开发者视角的完整设定，非玩家在游戏初期即可接触的信息。

---

## 一、核心背景

### 1.1 反重力时代

近未来的某一时期，大气层内反重力航行技术突然成熟，且**技术要求低到出人意料**。这项技术不是少数国家垄断的黑科技——它是一道门槛忽然消失之后涌进来的洪水。当一项技术足够便宜、足够简单、足够普及时，它改变的不是交通工具，是整个文明的空间逻辑。

大宗运输几乎全部转向反重力。传统公路、铁路、海运在一代人的时间内全面萎缩。那些因交通枢纽地位而繁荣的城市开始空心化。与此同时，"选址"的逻辑被彻底颠覆：

- 聚落不再需要靠近港口、铁路或高速公路出口。
- 新的聚居地可以出现在任何位置：山顶、峡谷底部、湖心岛。
- 出现了永久悬浮的空中社区——反重力技术使这一切成为现实。

### 1.2 来自地底的灾难

与大多数灾难来自天上的叙事不同，这个世界的灾难来自**地底**。

灾难的性质至今未有定论。目前已知的是：

- 地表逐渐变得不再宜居。
- 人类在经历了漫长的挣扎后，**正式承认了这一点**。
- 幸存者向空中迁移。

反重力技术在此刻从"重构生活的工具"转变为"生存的唯一保障"。

### 1.3 天堂时代

人类将空中世界称为**天堂**（Heaven），将自身称为**信徒**（Believers）。

在天堂时代，人类建造了庞大的空中文明体系：

- **座天使**级高空居住站成为了天堂的心脏。
- 数以万计的各类天使级飞船构成了维持文明运转的骨架。
- 一个以层级化天使命名体系为框架的空中社会，持续了相当长的时期。

### 1.4 缄默

然后——**信徒被抹去了**。

原因未知。过程未知。在某一个时间点之后，空中不再有人类的声音。系统依然在运转，飞船依然在执行最后的指令，但操作者消失了。只留下他们建造的那些天使，沉默地漂流在这片天空里。

这个时代，被称为**缄默**（The Silence）。

## 二、天使层级（飞船分类体系）

所有飞船统称"天使"。不同层级对应不同的职能与等级。层级名称倒置自古典天使学九阶体系。

| 层级 | 代号 | 职能描述 |
|------|------|----------|
| **炽天使** | Seraphim | 最高等级的战斗单元。全系统仅个位数存在。携带大量对地攻击设备、空优武装与机库。抵达战场的第一秒即预示战役结局。 |
| **智天使** | Cherubim | 指挥单元。负责战区协调、任务分配与信息中继。缄默时代中，是否存在仍受人类控制的智天使是一个核心谜团。 |
| **座天使** | Ophanim | 高空居住站。天堂时代的心脏，容纳了最多的信徒。天堂陷落后，大部分座天使去向不明。 |
| **能天使** | Powers | 浮空的工程母港。具备制作和维护下三级船只的能力。是天堂时代的工业脊梁。 |
| **力天使** | Virtues | 武装舰船。天堂时代的主力作战单元，火力与防护均衡。 |
| **主天使** | Dominions | 辎重船。负责后勤补给、货运与资源采伐。支撑天堂时代物流体系的基础层级。 |
| **权天使** | Principalities | 天基主动防御单位。部署于重要空域，执行区域封锁与拦截任务。 |
| **大天使** | Archangels | 对地攻击炮艇。拥有硕大无朋的船体，腹部挂载密集的对地攻击设备。跑团中的 destroyer 01（石云）即属于此层级。 |
| **天使** | Angels | 多功能舰船。带有模块化工作部位，可适配多种任务形态。跑团中的石鸟即属于此层级。 |

## 三、缄默谱系：天使的六种状态

信徒被抹去之后，留下的是六种不同状态的缄默天使。每一艘仍然存在于空中的船，都可以被归入这六类之一。

| 状态 | 代号 | 描述 |
|------|------|------|
| **空舟** | Empty Vessel | 系统完全正常，但不再有人操作。最常见的状态。船体是满的，但人是空的。跑团中的大天使石云即为此类。 |
| **堕天使** | Fallen | 系统已不能正常运行。有些还侥幸挂在天上，但已失去绝大部分功能；有一些已经坠毁。跑团中山谷中的大天使残骸即为此类。 |
| **追火者** | Fire Chaser | 不知为何在自主运行，仍在执行某段任务逻辑，但任务的上文早已失传。它们追着一个已经不存在的目标，像飞蛾扑火。 |
| **弦上人** | Stringsman | 受未知节点统一控制的飞船集群。它们的行动表现出明显的协调性，暗示背后存在一个仍在运行的智天使——但这个智天使本身是追火者，还是天意，无人知晓。 |
| **灯台** | Lighthouse | 指受人类直接控制的智天使。是否存在——这是一个贯穿游戏始终的核心疑问。 |
| **余响** | Reverberation | 系统大部分宕机，但偶尔仍会发出信号、回放日志片段、或在特定条件下短暂恢复部分功能。不是活的，也不是死的——是一段还在播放的录音。 |

## 四、命名机制（游戏体验）

玩家在游戏中首次接触重要实体（如大天使级飞船、关键设备、特殊地点）时，不会直接获知其天使层级代号或正式名称。系统会提示：

> "你可以在日志中为它命名。"

并提供一个**可选名称建议列表**（3–5 个备选），同时允许玩家自由输入。

示例——玩家第一次见到 destroyer 01 时可能看到的建议列表：

- 石云
- 铁砧
- 渡鸦
- 沉默之楔
- （自定义）

不同玩家的命名选择会形成完全不同的叙事基调。「石云」和「沉默之楔」指向的情感方向截然不同。游戏不预设正确答案。

## 五、待定问题

以下内容尚未完成构思，留待后续填充：

- **地底灾难的具体性质**——是地质灾难、生物威胁，还是人类自身行为的后果？
- **信徒被抹去的原因**——是整个天堂时代最大的悬念，也是玩家在游戏中逐步拼图的核心目标。
- **灯台是否存在**——如果存在，它在哪？如果不存在，"天意"这个概念对玩家意味着什么？
- **追火者与弦上人之间的关系**——独行的执拗者与被统一指挥的虔信者之间是否存在过渡形态？
