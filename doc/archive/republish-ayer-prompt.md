# 发布 Ayer 主题 Prompt

## 背景与约束

项目是部署于 GitHubPage 的 Hexo 博客，其所使用的主题 Ayer 原作者停止更新，由我以 "absorb & own" 方式融入项目。不过，因为在较长时间内，我曾无意再次分发此主题，导致开发过程中，网站与主题没有刻意进行解耦合。现在我打算发布主题，此 prompt 意在处理此问题。下文所有要求出具的报告或方案，均应置于doc\ayer。

## 主题功能调查

请您先清点主题目前能独立提供的功能，即如果立刻发布整个主题文件夹（themes\ayer），能为用户提供什么功能。如果不需要添加其他的功能、直接发布，会缺失什么关键文件；

## 站点功能调查

清点哪些站点功能可能应置于主题内而没有。

## 出具方案

出具解耦合方案。此外我将来有持续更新、维护主题的需求，不可能每次都跑一遍上述流程，请也告诉我业界惯例的做法。
